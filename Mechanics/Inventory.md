Inventory is divided into two main section: item ID and item quantity.
## Item ID
* Items: Can be found at:
	* `0x8014504c` in the RAM
* Weapons: Can be found at:
	* `0x801450cc` in the RAM
* Armor: Can be found at:
	* `0x8014514c` in the RAM
* Option: Can be found at:
	* `0x801451cc` in the RAM
## Item Quantity
* Items: Can be found at:
	* `0x8014524c` in the RAM
* Weapons: Can be found at:
	* `0x801452cc` in the RAM
* Armor: Can be found at:
	* `0x8014534c` in the RAM
* Option: Can be found at:
	* `0x801453cc` in the RAM
