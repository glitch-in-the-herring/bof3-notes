The player party's combatant data starts at `0x80145e90` in the RAM and is 320 bytes long for each combatant.

| Position | Description           | Value(s)                     | Note                                                                                                                                                                       |
| -------- | --------------------- | ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 5        | Combatant ID          | 8-bit unsigned integer       | Think of this as representing the order of the combatant (combatant 0, 1, 2, etc). This can't be used to tell who the combatant is                                         |
| 9        | Character ID          | 8-bit unsigned integer       | Think of this as representing who the combatant is (Ryu, Nina, etc). See [[Ordering]] for the list of IDs                                                                  |
| 116-120  | Name                  | ASCII null-terminated string |                                                                                                                                                                            |
| 128-129? | Status                | See [[Status]]               |                                                                                                                                                                            |
| 130      | Weapon                | 8-bit unsigned integer       |                                                                                                                                                                            |
| 131      | Shield                | 8-bit unsigned integer       |                                                                                                                                                                            |
| 132      | Helmet                | 8-bit unsigned integer       |                                                                                                                                                                            |
| 133      | Chestplate            | 8-bit unsigned integer       |                                                                                                                                                                            |
| 134      | Accessory 1           | 8-bit unsigned integer       |                                                                                                                                                                            |
| 135      | Accessory 2           | 8-bit unsigned integer       |                                                                                                                                                                            |
| 136-137  | Current HP            | 16-bit signed integer        |                                                                                                                                                                            |
| 138-139  | Current AP            | 16-bit signed integer        |                                                                                                                                                                            |
| 143      | Master                | See [[Masters]]              |                                                                                                                                                                            |
| 144-145  | Current max HP        | 16-bit signed integer        | Current max HP drops when the party member is knocked out mid-battle and is not revived during the battle. It is restored back to the true max HP after resting in an inn. |
| 146-147  | Current max AP        | 16-bit signed integer        |                                                                                                                                                                            |
| 148-149  | Effective PWR         | 16-bit unsigned integer      | This is the combatant's PWR after all equipment modifiers and buffs/debuffs.                                                                                               |
| 150-151  | Effective DEF         | 16-bit unsigned integer      | This is the combatant's DEF after all equipment modifiers and buffs/debuffs.                                                                                               |
| 152-153  | Effective AGL         | 16-bit unsigned integer      | This is the combatant's AGL after all equipment modifiers and buffs/debuffs.                                                                                               |
| 154-155  | Effective INT         | 16-bit unsigned integer      | This is the combatant's INT after all equipment modifiers and buffs/debuffs.                                                                                               |
| 159-167  | Effective resistances | 8-bit unsigned integers      | This is the combatant's resistances after all equipment modifiers and buffs/debuffs.<br>See [[Resistance]] for the order of resistances.                                   |
| 168      | Effective surprise    | 8-bit unsinged integer       |                                                                                                                                                                            |
| 169      | Effective reprisal    | 8-bit unsinged integer       |                                                                                                                                                                            |
| 170      | Effective critical    | 8-bit unsinged integer       |                                                                                                                                                                            |
| 171      | Effective dodge       | 8-bit unsinged integer       |                                                                                                                                                                            |
| 172      | Effective accuracy    | 8-bit unsinged integer       |                                                                                                                                                                            |
| 176-177  | True max HP           | 16-bit unsigned integer      |                                                                                                                                                                            |
| 178-179  | True max AP           | 16-bit unsigned integer      |                                                                                                                                                                            |
| 180-181  | Base PWR?             | 16-bit unsigned integer      |                                                                                                                                                                            |
| 182-183  | Base DEF?             | 16-bit unsigned integer      |                                                                                                                                                                            |
| 184-185  | Base AGL?             | 16-bit unsigned integer      |                                                                                                                                                                            |
| 186-187  | Base INT?             | 16-bit unsigned integer      |                                                                                                                                                                            |
| 191-199  | Base resistances?     | 8-bit unsigned integers      |                                                                                                                                                                            |
| 200      | Base surprise         | 8-bit unsinged integer       |                                                                                                                                                                            |
| 201      | Base reprisal         | 8-bit unsinged integer       |                                                                                                                                                                            |
| 202      | Base critical         | 8-bit unsinged integer       |                                                                                                                                                                            |
| 203      | Base dodge            | 8-bit unsinged integer       |                                                                                                                                                                            |
| 204      | Base accuracy         | 8-bit unsinged integer       |                                                                                                                                                                            |
| 284-285  | HP change             | 16-bit signed integer        | See [[HP-AP Change]]                                                                                                                                                       |
| 286-287  | AP change             | 16-bit signed integer        | See [[HP-AP Change]]                                                                                                                                                       |
| 288      | Heal status           | 8-bit unsigned integer       | See [[HP-AP Change]]                                                                                                                                                       |
| 292      | Action                | 8-bit unsigned integer       |                                                                                                                                                                            |
| 300      | PWR multiplier        | 8-bit signed integer         | See [[Stat Multiplier]]                                                                                                                                                    |
| 301      | DEF multiplier        | 8-bit signed integer         | See [[Stat Multiplier]]                                                                                                                                                    |
| 302      | AGL multiplier        | 8-bit signed integer         | See [[Stat Multiplier]]                                                                                                                                                    |
| 303      | INT multiplier        | 8-bit signed integer         | See [[Stat Multiplier]]                                                                                                                                                    |
| 312      | Focus counter         | 8-bit unsigned integer       | See [[Focus]]                                                                                                                                                              |
