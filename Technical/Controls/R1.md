The R1 button is used to change the azimuth and altitude of the [[Technical/Rendering/Camera|camera]] by holding it and pressing the d-pad buttons.
## Right
Pressing right increases the azimuth, which rotates the camera counterclockwise.
```
80199044 lh     $v0, 0x0000(a1) ; load current azimuth
80199048 nop
8019904c move   $a0, $v0
80199050 slti   $v0, 0x0355 ; check if the azimuth is at the allowed limit
80199054 beqz   $v0, 0x80199074
80199058 addiu  $a0, 0x0016 ; increase the azimuth
8019905c lui    $v0, 0x8015
80199060 addiu  $v0, -0x7e24
80199064 lhu    $v1, 0x0000(v0)
80199068 sh     $a0, 0x0000(a1) ; store new azimuth
8019906c addiu  $v1, 0x0016
80199070 sh     $v1, 0x0000(v0)
80199074 lui    $v0, 0x8014
80199078 lhu    $v0, 0x5aa4(v0)
8019907c nop    
80199080 andi   $v0, 0x8000
80199084 beqz   $v0, 0x801990c4
80199088 nop    
801990c4 addiu  $sp, 0x0020
801990c8 jr     $ra
801990cc nop    
```
The code can be found at `BIN/ETC/GAME.EMI: 00004050`
## Left
Pressing left decreases the azimuth, which rotates the camera clockwise
```
80199044 lh     $v0, 0x0000(a1) ; load current azimuth
80199048 nop
8019904c move   $a0, $v0
801990a0 slti   $v0, 0x00ab ; check if the azimuth is at the allowed limit
801990a4 bnez   $v0, 0x801990c4
801990a8 addiu  $a0, -0x0016 ; decrease the azimuth
801990ac lui    $v0, 0x8015
801990b0 addiu  $v0, -0x7e24
801990b4 lhu    $v1, 0x0000(v0)
801990b8 sh     $a0, 0x0000(a1) ; store the azimuth
```
The code can be found at `BIN/ETC/GAME.EMI: 000040a0`
## Up
Pressing up increases the altitude
```
80198ff4 lh     $v0, 0x0000(a1) ; load current altitude
80198ff8 nop    
80198ffc move   $a0, $v0
80199000 slti   $v0, -0x01c7 ; check if the altitude is at the allowed limit
80199004 beqz   $v0, 0x80199024
80199008 addiu  $a0, 0x0016 ; increase altitude
8019900c lui    $v0, 0x8015
80199010 addiu  $v0, -0x7e28
80199014 lhu    $v1, 0x0000(v0)
80199018 sh     $a0, 0x0000(a1) ; store new altitude
```
The code can be found at `BIN/ETC/GAME.EMI: 000040a0`
