The general table for key items differs from regular items. Each entry is 16 bytes long and can be broken down as such:

| Position | Description       | Value(s)                  | Note            |
| -------- | ----------------- | ------------------------- | --------------- |
| 0-11     | Name              | See the text guide        |                 |
| 12-13    | Description index | 16-bit unsigned integer   |                 |
| 14-15    | ?                 |                           |                 |
