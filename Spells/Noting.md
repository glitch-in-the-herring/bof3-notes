There are 12 Noting (or Nothing in the PAL version) spells that are (mostly) unused and non-functional.  

## Noting 1 

| ID           | Japanese Name                      | Description                           | AP Cost |
| ------------ | ---------------------------------- | ------------------------------------- | ------- |
| `0x19`<br>25 | しろはた *shirohata*<br>("White flag") | Grants immunity to normal attacks<br> | 0       |
Non-functional.
## Noting 2

| ID           | Japanese Name          | Description                         | AP Cost |
| ------------ | ---------------------- | ----------------------------------- | ------- |
| `0x50`<br>80 | ラーク *rāku*<br>("Lark") | Lower chance of meeting enemies<br> | 4       |
Non-functional. Shares its Japanese name with the Smoke spell from Breath of Fire II, which had the same function (were it to work).
## Noting 3

## Noting 11

| `0xd5`<br>213 | ミヤクリ *miyakuri*<br> | Use hourglass to stop time | 3       |
